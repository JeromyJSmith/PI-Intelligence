# PI Due Diligence Hub

A practitioner's resource hub for PI-adjacent due diligence — OSINT tools, people search platforms, regulatory databases, federal court records, and automation pipelines.

**Live site:** https://jeromyjsmith.github.io/pi-due-diligence/

## Pages

- **[index.html](index.html)** — Landing hub with pipeline overview, resource table, and feature summary
- **[osint-report.html](osint-report.html)** — Full OSINT Intelligence Report (9 sections, 40+ tools)
- **[due-diligence.html](due-diligence.html)** — Operational Due Diligence Guide (5-phase pipeline)

## Coverage

- FCRA, GDPR, CCPA legal compliance
- People search: BeenVerified, TruthFinder, Intelius, Instant Checkmate, free alternatives
- Company due diligence: OCCRP Aleph, OpenCorporates, Companies House, OpenSanctions
- Federal courts: PACER Case Locator
- Regulatory: IAPD, SEC enforcement, FINRA BrokerCheck
- AI-powered OSINT agents and GitHub toolboxes
- Automation roadmap and skills.sh packaging
